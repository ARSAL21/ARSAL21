## Projects

### Web Store — E-Commerce Platform
Full-stack e-commerce system built with strict architectural separation
for scalability and maintainability.

Implemented Service Layer with DTOs, Redis-backed queue system for
background processing, Moota payment gateway, courier API integration,
and automated regional data fetching covering 90k+ administrative

### SIM-PDAM — Rural Water Utility Management System
Web application built to digitize water utility billing and consumption
tracking for rural communities.

Implemented a contract-handover system for relocating residents,
interactive consumption monitoring via charts, automated billing logic,
and WhatsApp notifications via Fonnte API.

`Laravel` `Livewire v3` `Filament` `MySQL`

[View Repository →](https://github.com/ARSAL21/SIM-PDAM)

### Smart-Letter — Village Document Management System
Platform to automate administrative document generation for village
offices, eliminating manual paperwork for residents.

Implemented dynamic placeholder detection and mapping in document
templates using PHPOffice, wrapped in database transactions for strict
data integrity. Granular RBAC with custom rule enforcement per document
type.

`Laravel` `Filament` `Tailwind CSS`

[View Repository →](https://github.com/ARSAL21/Smart-Letter)
